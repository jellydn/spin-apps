## Prerequisites
- [Fast, disk space efficient package manager | pnpm](https://pnpm.io/)
- [Install Spin](https://developer.fermyon.com/spin/install)
- [TinyGo](https://tinygo.org/getting-started/install/macos/)

## Usage

```sh
spin plugin install js2wasm
make build
make dev
```

## Deployment

```sh
make deploy
```
